# Little Lemon Restaurant Booking App

A React web application that allows users to reserve a table at the Little Lemon restaurant.

---

## Features

- Table reservation form
- Form validation
- Dynamic booking times using React state
- LocalStorage persistence
- Unit testing with Jest
- Responsive design
- Accessible semantic HTML

---

## Tech Stack

- React
- JavaScript
- CSS
- Jest
- React Testing Library

---

## Run the Project

Install dependencies:

npm install

Start the app:

npm start

Run tests:

npm test
